# 📖 less - View file contents page by page

| Key | Description |
|---|---|
| `Space` | Page down (move forward one screen) |
| `b` | Page up (move backward one screen) |
| `g` | Go to the beginning of the file |
| `G` | Go to the end of the file |
| `/pattern` | Search forward for pattern |
| `?pattern` | Search backward for pattern |
| `n` | Go to next search result |
| `N` | Go to previous search result |
| `q` | Quit less |
| `:n` | Open next file (if multiple files) |
| `:p` | Open previous file (if multiple files) |
| `v` | Open current file in text editor |
| `h` | Display help menu |

💡 **Tip:** Use `Ctrl + C` or `q` anytime to exit less and return to the terminal.
