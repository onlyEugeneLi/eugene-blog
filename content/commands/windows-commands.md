# Useful NPM Config Commands

```
npm config list
#Shows the current config in Terminal
```

```
npm config get cafile
#Get the value of a specific npm config variable
```

```
npm config set ca="C:\Users\P-number\Nationwide_Root_CA2.crt"
#Set the value of a specific npm config variable
```

```
npm config edit
#Allows editing of the config file, generally found at C:\Users\P-number\.npmrc or wherever you have specified it.

#It is worth searching Windows Explorer for .npmrc to find any other copies that the system may be pointing at.
```
